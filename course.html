HTML

Description of Course 1.

  • HTML (HyperText Markup Language):
  • HTML is the standard markup language for creating web pages.
  • It provides a structure to web content by using tags to define elements like headings, paragraphs, images, links, etc.
  • It is not a programming language but a markup language used to organize and format content on the web.

    • CSS

    Description of Course 2.

  • CSS (Cascading Style Sheets):**
  • CSS is a stylesheet language used for describing the presentation of a document written in HTML.
  • It controls the layout, design, and appearance of web pages.
  • CSS allows developers to separate the structure of a web page (HTML) from its style, enhancing the flexibility and maintainability of websites.

    • JAVASCRIPT

    Description of course 3

  • JavaScript:
  • JavaScript is a versatile programming language that enables interactivity and dynamic content on websites.
  • It runs on the client side, allowing manipulation of the Document Object Model (DOM) and handling user events.
  • JavaScript is widely used for creating responsive and interactive web applications.
